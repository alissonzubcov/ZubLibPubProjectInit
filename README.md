# ZubLib

Biblioteca Angular de componentes customizados.

## Scripts principais

- `ng serve` — Inicia o servidor de desenvolvimento.
- `ng build` — Gera o build de produção.
- `npm run test:karma` — Executa testes unitários com Jasmine/Karma.
- `npm run test:jest` — Executa testes unitários com Jest.

## Estrutura

- Componentes em `src/app/`
- Estilos globais em `styles/`

## Requisitos

- Node.js e npm
- Angular CLI 21+

## Testes

- Testes unitários: Jasmine/Karma ou Jest (ambos configurados)
